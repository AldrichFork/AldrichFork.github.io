小结

indexOf()方法

indexOf()方法可返回某个指定的字符串值在字符串中首次出现的位置
searchvalue参数是指定的字符串值，是可选的整数参数。
start规定在字符串中开始检索的位置。它的合法取值是 0 到 string Object.length - 1。如省略该参数，则将从字符串的首字符开始检索。
返回值类型：number

substring()方法

substring() 方法用于提取字符串中，结余两个指定下标之间的字符，返回的子串包括开始出的字符，但不包括结束处的字符。
语法：string.substring(from,to) .
from和to都为非负整数
from为提取的子串的第一个字符在string对象中的位置
to为要提取的子串的最后一个字符在string对象中的位置+1，如果不写to参数，那么返回的子串会一直到字符串的结尾。

parseInt() 函数

parseInt() 函数可解析一个字符串，并返回一个整数。
语法 parseInt(string,radix) .
parseInt() 函数可解析一个字符串，并返回一个整数。
当参数 radix 的值为 0，或没有设置该参数时，parseInt() 会根据 string 来判断数字的基数。
当忽略参数 radix , JavaScript 默认数字的基数如下:

  • 如果 string 以 “0x” 开头，parseInt() 会把 string 的其余部分解析为十六进制的整数。
  • 如果 string 以 0 开头，那么 ECMAScript v3 允许 parseInt() 的一个实现把其后的字符解析为八进制或十六进制的数字。
  • 如果 string 以 1 ~ 9 的数字开头，parseInt() 将把它解析为十进制的整数。
